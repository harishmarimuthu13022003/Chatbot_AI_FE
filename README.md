# AI Chat Application - Frontend

This is the React frontend for the AI Chat Application.

## Tech Stack
- **React** (Vite)
- **Axios** (API requests)
- **Vanilla CSS** (Premium Dark Mode UI)
- **Lucide React** (Icons)

## Setup Instructions

1. Install the required Node.js dependencies:
   ```bash
   npm install
   ```
2. Start the Vite development server:
   ```bash
   npm run dev
   ```
3. The application will be accessible at `http://localhost:5173`. Make sure the FastAPI backend is running simultaneously on `http://localhost:8000`.
